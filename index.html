Visual AI Workflow Automation for Chrome

Prerequisites: Chrome 138+ with Chrome's Built-in AI APIs enabled. Join the Chrome Built-in AI Early Preview Program.

Installation

Download: GitHub Repository (clone or download ZIP)

  1. Extract/clone the repository and locate the build-unpacked folder
  2. Open chrome://extensions in Chrome
  3. Enable Developer mode (top-right toggle)
  4. Click Load unpacked
  5. Select the build-unpacked folder

What It Does

Flosheet AI is a visual workflow builder for Chrome's built-in AI APIs. Users drag and connect nodes to create custom AI workflows (summarize, translate, etc.) that run entirely client-side.

Key Features

  • Visual node-based workflow editor (no coding required)
  • Plugin architecture - each node is extensible
  • Client-side execution using Chrome's Summarizer, Translator, and Prompt APIs
  • Context menu integration for quick access
  • Privacy-first: all processing happens locally

APIs Used

  • Summarizer API - Extract insights from web content
  • Translator API - Multilingual translation capabilities
  • Prompt API - Custom text generation with Gemini Nano

Use Cases

  • Power users: Visual automation without coding
  • Developers: Rapid prototyping for AI extension ideas
  • Content creators: Multilingual content processing workflows

Technical Details

Stack: React + TypeScript, React Flow, Chrome Extension APIs

Architecture: Plugin-based node system, context-aware execution, headless mode support
